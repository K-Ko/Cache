Class MemCache

Cache class using MemCache server

The following settings are supported:

  • Token : used to build unique cache Ids (optional)
  • MemCache : host[:port] (optional) default: localhost:11211

If MemCache is not installed, gMemCache is used, a purely implementation of a MemCache client in PHP. http://www.phpclasses.org/package/4094-PHP-memcache-client-in-pure-PHP.html
